Glosario de Términos

  1. <a>: Es el elemento que nos permite enlazar nuestros documentos HTML entre sí, o a otras webs externas a la nuestra.
  2. <body>: Este elemento define el cuerpo principal del documento HTML que estamos editando.
  3. <div>: Es el elemnto más utilizado en HTML 4 y sirve para agrupar contenidos.
  4. <em>: Dentro de los elementos para el resaltado de texto, es aquél que nos permite marcar un texto con una mayor importancia, pero desde el punto de vista semántico y no desde visual.
  5. <body>: Contiene todo el contenido visible en la página web, como el texto, las imágenes y los enlaces.
  6. <h1> - <h6>: Define los encabezados o títulos de una página web, siendo el h1 el de mayor importancia.
  7. <footer>: Nos permite englobar el contenido que corresponde al pie de nuestra pagina web. Por ejemplo la información de contacto de nuestra empresa.
  8. <html>: Es el tag que define el comienzo y fin de un documento.
  9. <img>: Inserta una imagen en la página web.
  10. <hr>: Este elemento nos permite insertar una linea horizontal que establece una división en el texto.
